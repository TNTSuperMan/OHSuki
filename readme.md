# Brainf**k版コードについて
Brainf**kはUTF-8などのASCII以外の文字が出力できない言語なため、英語で"suki"を100回出力するようにしました。

また、このコードは[BrainHack](https://github.com/TNTSuperMan/BrainHack)というプログラムが生成しました。
さすがにBrainf**k生で開発は勘弁です。
